# winapi_utils
Набор учебных проектов на C/C++, направленных на изучение различных winapi механизмов, а так же разработке windows kernel mode драйверов файловой системы

Каждая директория содержит файл с описанием проекта


Некоторые проекты содержат в названии сокращенное наименование предмета, в рамках которого выполнялось задание

В будущем, проекты из данного репозитория могут использоваться для разработки других проектов.
